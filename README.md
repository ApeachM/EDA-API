# EECE667-termProject
Placement Implementation in Cpp with OpenDB API

## How to build

```shell
git clone https://github.com/ApeachM/EECE667-termProject.git
cd EECE667-termProject
```

```shell
mkdir build
cd build
cmake ..
make
./Placer
```
